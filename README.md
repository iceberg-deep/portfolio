# Portfolio

Single-page professional portfolio for Justen P. — Detection-Informed Penetration Tester.

Static, dependency-free HTML served by GitHub Pages. No build step.

- **Live:** <https://iceberg-deep.github.io/portfolio/>
- **Edit:** everything is in `index.html` (styles inline in `<head>`).

## Deploy

Pushes to `main` publish automatically once GitHub Pages is enabled
(Settings → Pages → Source: `main` / root).

## Notes

- Contact routes through LinkedIn / GitHub only — no email or phone anywhere on
  the page or in the source, by design.
- `assets/og.png` (1200×630) is referenced for social link previews — drop one in
  to get a thumbnail when the link is shared on LinkedIn/X.
- "Save as PDF" uses the browser print dialog; a print stylesheet renders a clean
  light-on-white version for a one-page résumé export.
